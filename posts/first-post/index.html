初投稿

December 12, 2021 One-minute read

技術系の学習をアウトプットするために本サイトを作成してみました。

気軽に投稿できるブログで開始しようと思い色々なブログサービスを調べましたが、どれも自分にはしっくりこなかったため前々から気になっていたHugoで作成してみました。

簡単にサイト作成ができるのでオススメです。

Theme はhugo-coderを使ってます。

以下導入手順を簡単にメモ

Hugo をインストール(for Mac) 

brew install hugo

サイトの作成 

Hugo new site my-site

テーマのインストール 

git submodule add https://github.com/luizdepra/hugo-coder.git themes/hugo-coder

テーマはこちらから選びました。

config.toml に theme を追記 

theme = "hugo-coder"

サーバー起動 

hugo server

http://localhost:1313 にアクセスするとインストールしたテーマの内容が表示されます。

簡単にサイト作成ができました。 今後も色々弄ってみて気づきがあれば記事にしてみようと思います。
